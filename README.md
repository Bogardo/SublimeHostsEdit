SublimeHostsEdit
================

A Sublime Text 2/3 plugin to ease the process of editing your hosts file.

Features
--------

### Quick Open ###
This plugin adds a hotkey for opening your hosts file.

* Windows: `ctrl+shift+h`
* Linux: `ctrl+shift+h`
* OSX: `md+shift+h`

There is also an extra menu option under `Tools`
